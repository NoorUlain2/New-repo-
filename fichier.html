Les commandes unix.

Comme commande unix, j'ai appris :

  • mkdir + 'nom du fichier' sert a creer un nouveau dossier
  • cd + 'nom du fichier'permet de se deplacer vers le fichier
  • rmdiv + 'nom du fichier' permet de supprimer un dossier
  • ls permet de lister le contenu
  • touch + 'nom du fichier' sert a creer un nouveau fichier
  • cd permet de revenir dans le fichier precedant
